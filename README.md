# YouTube Channel Analytics Project

## Overview
Analysed performance data from my YouTube channel (2,500+ followers)
to identify top-performing content, engagement patterns, and viewer
retention trends using Microsoft Excel.

## Tools Used
- Microsoft Excel (pivot tables, formulas, charts, data cleaning)
- YouTube Studio Analytics (data export)

## What I Did
- Exported channel analytics data across 3 files (per-video stats,
  daily trends, and channel totals)
- Cleaned raw CSV data: fixed number formatting, removed errors,
  standardised date columns
- Created 3 calculated metrics: Engagement Rate, Avg Watch Time
  per View, and a weighted Performance Score
- Built 3 charts and a summary dashboard in Excel

## Key Findings
- Total engaged views: 97,876
- Daily average views: 84
- Best single day: 8,528 views
- Top video: Para #03 Urdu Translation with 10,548 views
- Average CTR across all videos: 9.44%
- Channel views declined steadily from 1-Jan-2023 — opportunity
  to revive with new content strategy
